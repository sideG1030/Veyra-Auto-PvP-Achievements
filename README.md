# Veyra-Auto-PvP-Achievements
Tampermonkey userscript for automating Veyra PvP battles to get Achievements.

•Select your class and your advanced class;

•Select which skill you want the script to spam:
    •The script will use your selected skill if it can, otherwise, it will use Slash:
    •If you select the ultimate advanced skill, the first attack will be Power Slash to not waste Match Tokens;
•If Consecutive battles are selected, the script will keep running until you are out of tokens:
      •You can specify how many times to use the selected skill. Once the scrit uses the selected skill that amount of times, it will stop Consecutive battles after finishing the current battle;
      •If you run out of PvP tokens, the script will remain idle until server hour passes, after which the scrip auto-resumes.
